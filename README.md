The steps to make github work:

First you will ‘add’ the file to github. Use:
git add FILENAME

Then you must commit:
git commit -m “MESSAGE”

Then you push it to the server:
git push

TO REMOVE

Remove command
git rm FILENAME

Then you must commit:
git commit -m “MESSAGE”

Then you push it to the server:
git push
